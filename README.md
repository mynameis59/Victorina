# Quiz Game

## Description
Console quiz application written in C#.
The program asks 3 questions and shows the final score.

## How it works
- The program displays a question
- The user enters an answer
- The program shows whether the answer is correct or not
- At the end, the total score is displayed

## Technologies
- C#
- .NET Console Application

## How to run
1. Open the project in Visual Studio
2. Press Start (F5)
